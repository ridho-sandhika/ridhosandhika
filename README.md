# 🚀 Ridho Sandhika - Modern 3D Personal Portfolio

Sebuah website portofolio pribadi interaktif yang didesain dengan konsep **Modern 3D** dan **Glassmorphism**. Website ini tidak hanya menampilkan tautan (link tree), tetapi juga menghadirkan fungsionalitas cerdas dan tampilan visual yang memanjakan mata.

🌐 **Live Demo:** [https://ridho-sandhika.github.io/portfolio]

## ✨ Fitur Utama

* **🎨 Desain 3D & Glassmorphism:** Efek visual kedalaman (depth), pencahayaan sudut, dan pantulan kaca yang elegan pada UI.
* **📍 Deteksi Lokasi Otomatis:** Menggunakan Geolocation API untuk mendeteksi lokasi pengguna saat mengakses website.
* **🕌 Real-Time Jadwal Sholat:** Terintegrasi dengan Aladhan API untuk menampilkan waktu sholat berikutnya secara *real-time* dengan hitung mundur (countdown), menyesuaikan lokasi pengguna.
* **🎵 Interactive Intro Sound:** Dilengkapi dengan kontrol pemutar musik latar yang mulus.
* **📱 Responsif Penuh (Mobile-Friendly):** Tampilan menyesuaikan dengan sempurna baik di layar *smartphone* maupun *desktop*.
* **⚡ Tactile Buttons:** Animasi tombol interaktif yang terasa seperti ditekan secara fisik.

## 🛠️ Teknologi yang Digunakan

* **HTML5** - Struktur utama website.
* **CSS3** - Animasi *keyframes*, *variables*, efek 3D (*inset shadow*), dan *glassmorphism*.
* **Vanilla JavaScript** - Logika interaktif, pemanggilan API, dan manipulasi DOM (Tanpa *framework* tambahan).
* **[Aladhan API](https://aladhan.com/prayer-times-api)** - Layanan penyedia data waktu sholat.
* **[OpenStreetMap / Nominatim API](https://nominatim.org/)** - Untuk melakukan *Reverse Geocoding* (mengubah titik kordinat menjadi nama kota).

## 🚀 Cara Menjalankan Secara Lokal

1.  Clone repository ini:
    ```bash
    git clone [https://github.com/ridho-sandhika/nama-repository-kamu.git](https://github.com/ridho-sandhika/nama-repository-kamu.git)
    ```
2.  Buka folder proyek.
3.  Jalankan file `index.html` menggunakan browser (Sangat disarankan menggunakan ekstensi seperti **Live Server** di VS Code agar fitur lokasi & API berjalan maksimal tanpa *CORS error*).
